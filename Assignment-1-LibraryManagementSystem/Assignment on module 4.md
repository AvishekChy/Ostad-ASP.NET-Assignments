#### **Assignment on module 4**

✅ Create a simple Library Management System Console App.



**Requirements:**



✅1. Create a **base class** called "**LibraryItem**" with the following:



&nbsp;  ✅ Properties: "Title" (string), "ItemId" (int), "IsAvailable" (bool)



&nbsp;  ✅ A constructor to initialize these properties



&nbsp;  ✅ A virtual method "DisplayInfo()" that prints the item details



**2. Create two derived classes:**



&nbsp;  ✅ "**Book**" class that inherits from "LibraryItem"



&nbsp;    ✅ Additional property: "Author" (string)



&nbsp;    ✅ Override the "DisplayInfo()" method to include author information



&nbsp;  



&nbsp;  ✅ "**Magazine**" class that inherits from "LibraryItem"



&nbsp;    ✅ Additional property: "IssueNumber" (int)



&nbsp;    ✅ Override the "DisplayInfo()" method to include issue number



**3. In the "Main" method:**



&nbsp;  ✅ Create 2 Book objects and 1 Magazine object



&nbsp;  ✅ Display information for all items



&nbsp;  ✅ Demonstrate borrowing an item (set "IsAvailable" to false)
